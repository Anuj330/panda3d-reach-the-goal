# Reach the Goal

A simple beginner-friendly 3D Panda3D game.

## Run locally

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 main.py
```

## Controls

- `W A S D` or arrow keys: move
- `R`: restart
- `Esc`: quit

## Goal

Move the panda to the yellow goal marker.

When you reach it, the game shows a success message.
